# CollectionsDemo

A simple and efficient Java program that demonstrates the capabilities of the `java.util.Collections` utility class. This project showcases essential collection operations such as sorting and finding extreme values within a list of elements.

---

## 🚀 Features

The application processes a list of integers to demonstrate the following core functionalities:
* **Ascending Sort:** Reorders the elements from smallest to largest using `Collections.sort()`.
* **Descending Sort:** Reorders the elements from largest to smallest using `Collections.sort()` paired with `Collections.reverseOrder()`.
* **Minimum Value Lookup:** Quickly retrieves the smallest element in the collection using `Collections.min()`.
* **Maximum Value Lookup:** Quickly retrieves the largest element in the collection using `Collections.max()`.
